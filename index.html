Question 4:
                    Print output:
                    01 var x = 21;
                    02 var girl = function () {
                    03     console.log(x);
                    04     var x = 20;
                    05 };
                    06 girl ();

line 03: undefined ; console.log(x); tries to log the value of the local variable x. However, due to hoisting, the local variable x is declared in the function scope, but it hasn't been assigned a value yet. So, at this point, the value of the local x is undefined.
